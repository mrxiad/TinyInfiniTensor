## 3. 作业必须理解的 ONNX 语义

1. Transpose：`permute` 重排规则
2. Clip：逐元素，shape 不变
3. Cast：shape 不变，dtype 变化
4. Concat：指定维拼接
5. Broadcasting：多向广播（从末尾维对齐）
6. Matmul：最后两维做矩阵乘，前置 batch 维广播

参考入口已在 `docs/训练营作业介绍.md` 每题代码块中给出。
