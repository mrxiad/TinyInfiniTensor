## 2. 代码改动

文件：`src/utils/operator_utils.cc`

函数：`infer_broadcast(const Shape &A, const Shape &B)`

实现内容：

1. 取 `rank = max(rankA, rankB)`。
2. 从末尾维度开始对齐（trailing alignment）。
3. 对每个对齐轴：
   - 缺失轴按 `1` 处理（例如标量、低 rank 张量）
   - 校验 `dimA == dimB || dimA == 1 || dimB == 1`
   - 输出轴长度为 `max(dimA, dimB)`
4. 返回输出 shape。

同时增加健壮性断言：

- `dimA > 0 && dimB > 0`

## 3. 算法语义说明

## 3.1 ONNX 多向广播规则（本实现对应）

对于两个 shape：

1. 从最后一维开始逐轴对齐。
2. 每个轴合法条件：
   - 两边相等，或
   - 其中一边为 1，或
   - 一边不存在（等价于 1）
3. 输出轴长度取两者较大值。

## 3.2 标量与不同 rank 的处理

- 标量 `{} `被视为“所有轴都是 1”。
- 低 rank 张量在高位自动补 1。

例如：

1. `{2,3,4,5}` 与 `{}` -> `{2,3,4,5}`
2. `{2,3,4,5}` 与 `{5}` -> `{2,3,4,5}`
3. `{1,4,5}` 与 `{2,3,1,1}` -> `{2,3,4,5}`

## 6. 复杂度分析

设 `R = max(rankA, rankB)`：

1. 时间复杂度：`O(R)`
2. 空间复杂度：`O(R)`

## 7. 健壮性与错误策略

当前策略：

1. 不可广播的轴组合直接 `IT_ASSERT` 失败。
2. 非正维度直接断言失败。

优点：

- 错误在构图期尽早暴露，避免 runtime 隐性崩溃。
